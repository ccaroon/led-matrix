## info_panel
Various modules that can be displayed on an LED Matrix and updated separately.

Currently includes:

* Digital Clock
* Weather (Temperature, Humidity, Hi/Low Temp)
* Binary Clock
* Fibonacci Clock
* ISS Tracker
* Daily Messages
