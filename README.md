# LED Matrix
LED Matrix Playground


## playground
Experiments.

## game_of_life
Conway's Game of Life

## info_panel
Various modules that can be displayed on an LED Matrix

Currently includes:

* Digital Clock
* Weather (Temperature, Humidity, Hi/Low Temp)
* Binary Clock
* ISS Tracker
* Daily Messages
